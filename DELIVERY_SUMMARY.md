## 🚀 How to Get Started (3 Steps)

### Step 1: Environment Setup (2 min)
```bash
cd e:/projects_6/longevity-challenge
npm install
cp .env.example .env.local
# Edit .env.local with your Supabase credentials
```

### Step 2: Supabase Setup (5 min)
1. Create account at supabase.com
2. Create new project
3. Copy Project URL & Anon Key to .env.local
4. Paste supabase_schema.sql into SQL Editor and run
5. (Optional) Run: `python3 insert_questions.py`

### Step 3: Launch App (1 min)
```bash
npm start
# Press 'i' for iOS or 'a' for Android
```

**Total setup time: 8 minutes**
